
# Local Cloud Storage Project

## How to Run

1. Make sure you have Python 3 installed.
2. Install Flask:
   ```
   pip install flask
   ```
3. Run the app:
   ```
   python app.py
   ```
4. Open your browser and go to:
   ```
   http://127.0.0.1:5000
   ```

## What it Does
- Upload files via a simple HTML form.
- Save files locally in the 'cloud_storage' folder.
- Retrieve files by URL: /files/<filename>
